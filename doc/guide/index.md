# Introduction

ConfigKits is a configuration-driven rendering framework that allows you to build dynamic applications from JSON configurations.

## What is ConfigKits?

ConfigKits separates UI structure and behavior from application code. Instead of hardcoding components, you define them in configuration files that can be updated without redeploying your application.

## Key Concepts

1. **Configuration-First**: Define UIs declaratively in JSON
2. **Schema Validation**: Type-safe configurations with Zod
3. **Component Registry**: Map configuration types to React components
4. **Feature Flags**: Conditional rendering and rollout control
5. **Lifecycle Hooks**: Extend functionality at key points

## Use Cases

- Dynamic UI generation
- Feature flag-driven development
- A/B testing interfaces
- Multi-tenant applications
- Micro frontend orchestration
