# Installation

ConfigKits is distributed as multiple npm packages. Install only what you need.

## Core Package

```bash
npm install @configkits/core
```

## React Renderer

```bash
npm install @configkits/react
```

## Feature Flags

```bash
npm install @configkits/flags
```

## All Packages

```bash
npm install @configkits/core @configkits/react @configkits/flags
```

## Requirements

- Node.js >= 18.0.0
- For React renderer: React >= 18.0.0
